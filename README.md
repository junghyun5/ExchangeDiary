# ExchangeDiary

## Project Summary
ExchangeDiary is an Android journaling application prototype that allows users to write, organize, and view diary entries.  
This project is designed as a senior-level computer science project demonstrating software development, object-oriented design, data handling, parsing, and user interface development.

## Features
- Create diary entries
- View a list of diary entries
- Read diary details
- Search diary entries
- Parse simple diary tags such as mood or topic

## Tech Stack
- Kotlin
- Android Studio
- Jetpack Compose
